关于本系统

功能特性

  • 🎯 智能推荐 - 基于最新胜率数据的推荐算法
  • 📊 数据驱动 - 实时从 Tacticians.gg 获取数据
  • 🔄 羁绊分析 - 自动识别和分析阵容羁绊
  • ⚡ 快速响应 - 毫秒级推荐生成

推荐算法

系统综合考虑以下因素：

  • 胜率权重 (50%) - 卡牌的总体胜率
  • 羁绊权重 (30%) - 与当前阵容的羁绊匹配度
  • 名次权重 (20%) - 卡牌的平均排名

数据来源

  • 📈 Tacticians.gg - 主要胜率数据源
  • 🎮 官方 Riot API - 卡牌信息和设定
  • 💾 本地缓存 - 自动更新缓存(24小时)
